# Lake-Filling Event Analysis

Analyzes lake-filling events in Australian ephemeral lakes using satellite observations 
(Digital Earth Australia) and gridded rainfall data (AGCD).

## Getting Started

1. Download the required data files:
```bash
   pip install gdown
   python download_data.py
```

2. Run the analysis:
```bash
   python LakesCombined_2025.py
```


## Contents
- `LakesCombined_2025.py` — main analysis script
- `download_data.py` — downloads required data from Google Drive
- `data/` — data directory (populated by download script)
- `output/` — figures generated by the analysis script


## Configuration
The script can be adapted for different lakes and time periods by editing the 
CONFIGURATION section at the top of `LakesCombined_2025.py`. Key parameters include:

- `Lake` — lake identifier (e.g., `'LW'` for Lake Woods)
- `ds_start` / `ds_end` — analysis period
- `window_size_daily` — rolling window size (default: 112 days)
- `lower_threshold_daily` / `higher_threshold_daily` — rainfall thresholds (default: 600mm, 800mm)

Note: The provided data files are for Lake Woods (LW) only.
